# Lesson Notes

This folder contains companion notes for the runnable lesson files.

Use these notes when you want:

- a clearer mental model for the topic
- extra context before trying the practice prompt
- common beginner mistakes to watch for
- a quick review after running the lesson

The notes are intentionally short. They are meant to deepen understanding
without turning each lesson into a textbook chapter.
